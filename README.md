# Tank Souls

Tank Souls - souls-like arcade shooter. The game features is boss fights, there is 3 bosses with different mechanics and estus flask. Built with React, TypeScript, and Canvas.

## How to Run Locally

**Prerequisites:** `npm, Node.js`


1. Clone repo: `git clone https://github.com/rocksJS/Tank-souls-beta.git`
2. Install dependencies:
   `npm install`
3. Run the app:
   `npm run dev`
4. Open link in terminal: `http://localhost:3000/`
